Superscript and Subscript

Superscript is used to display text slightly above the normal line.



1. (x + y)2 = x2 + y2 + 2xy

2. (a - b)2 = a2 - 2ab + b2

3. a2 + b2 = (a + b)2 - 2ab

4. (a + b + c)2 = a2 + b2 + c2 + 2ab + 2bc + 2ca

5. (a + b)3 = a3 + b3 + 3ab 2 + 3a2b

6. 2H2 + O2 → 2H2O

7. S8 + 12O2 → 8SO3

8. 2Na + 2H2O → 2NaOH + H2
